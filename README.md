# playground-virtual-app
Playing around with some `virtual-*` tech. Included is:
- [hyperx](https://github.com/substack/hyperx) - tagged template string virtual
  dom builder
- [virtual-app](https://www.npmjs.com/package/virtual-app) - unidrectional
  state management with virtual-dom
- [virtual-dom](https://github.com/Matt-Esch/virtual-dom) - virtual dom and
  diffing algorithm

## development
Install `budo` globally:
- [budo](https://www.npmjs.com/package/budo) - browserify server for rapid
  prototyping

Then run
```sh
$ ./start
```
